# harriet_front_end
###harriet前端 ，使用jquery ui等插件和ajax技术
<p>注册表单：form.js，具有提示输入tooltip和日历datepicker还有错误提示validate.js，邮箱用了自动补全autocomplete.
登录后会用cookie保存用户信息。
搜索的自动补全功能，返回的是由ajax取出数据库中数据:
从用户输入的第一的字符开始传给php文件，然后再从数据库中选取符合的问题返回到页面</p>
